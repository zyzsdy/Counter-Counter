Help

What is Counter Counter?

Counter Counter is a simple tool that you can count something and save the result. When add a counter and click, its number will increase. That you could count anything one by one through this tool.

How to Start?

While you start the app, a empty counters page will shown. Click the gray PLUS button or use menu "Counter -> New Counter" then a new counter will show out. Next, you will start your count.

  • CLick the number to increase.
  • Press "Reset" then number wiill reset to 0
  • Press the key shown at the bottom of the counter could also increase.
  • Hold SHIFT and click the number to decrease. Press SHIFT + Hot Key can do so.
  • Doubleclick the "×" at the top left of counter will delete it.
  • Click the "Title" then you could rename a counter.

Save as a file

You can save the result of your all counters this page. Use "File -> Save" save to local disk. Or "File -> Save as" to save in another copy.

Menu "File -> Open File" could load a Counters Page and display all the Counters. You can also modify it

When current Counters Page have modified, a circle dot "â—" will shown before the filename at title bar. Until you save it again througe "File -> Save" or "Save as" command. So, this dot can remind you the current save status.

Hot keys

There are some hot keys could be used for some common commands.

Every counter will try to get a Hot Key. If a counter got a hot key, a key name will be displayed in the key tip in the bottom left of the counter. If not, "---" will be displayed.

If you want to change the hot key, just click the key tip and press a new key. If new hot key is available, the key name will be displayed at the key tip. Press "BACKSPACE" will delete the hot key that the counter had.

There are some global hot keys for menu commmands.

  • New Page ----------- Ctrl + N
  • Open File ---------- Ctrl + O
  • Save --------------- Ctrl + S
  • Save as ------------ Ctrl + Shift + S
  • Exit the app ------- Alt + X

Always on Top

If you want show the window at the top of any other active windows that you can keep watch over the count easily. You could use the "Help -> Always on Top" command. When a checkbox have been checked, the window of this app will shown at the top of all windows. Use the command again will convert it to normal status.
